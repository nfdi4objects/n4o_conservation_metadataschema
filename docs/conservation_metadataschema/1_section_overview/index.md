# Sektions Überblick

Übersicht über alle Sektionen des Metadatenschemas. Die Sektionen spiegeln verschiedene Prozesse im Umgang mit Kunst- und Kulturgut im Kontext konservatorisch-restauratorischer Maßnahmen wider. Jede Sektion enthält eine Reihe von Metadatenelementen, die die Informationen zu einem Objekt beschreiben.
